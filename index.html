Formas de implementar estilos css.

  • Inline
  • Interno
  • Externo

Inline

Las instrucciones css en línea son aquellas que se escriben directamente a la etiqueta en el archivo HTML en lugar de utilizarlas en un archivo CSS externo.

Interno

Las diferentes propiedades y valores se pueden poner en una misma línea o en distintas líneas según se prefiera (siempre separados mediante punto y coma).

Externo

Está basada en declarar los estilos CSS en un archivo independiente que puede servir como referente para dotar de estilos a decenas, cientos o miles de archivos html, que únicamente deberán invocar el nombre de archivo utilizando una sintaxis específica
